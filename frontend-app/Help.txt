Help info:
install Node.js

• node -v

• npm -v

• npm install -g @angular/cli

Check Angular CLI version
• ng version

Create a new Angular application:
• ng new my-app

Start the application:
• ng start

Generage TestService without tests in the services folder, register it to the dependencies:
• ng g s services/test --skip-tests

Generage FirstComponent without tests in the components folder, register it to the dependencies:
• ng g c components/first --skip-tests

Generage FilterProductsPipe without tests in the pipes folder, register it to the dependencies:
• ng g p pipes/filter-products --skip-tests
